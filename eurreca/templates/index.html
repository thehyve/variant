Welcome to the EURRECA Variant database

The Variant database is an online repository that stores information on genotype, phenotype and micronutrient interactions.

You can use the database to

  • store studies that describe genotype, phenotype, micronutrient interaction data
  • search for known micronutrient-phenotype, interactions
  • explore which biological processes micronutrients are involved in
